# 📺 DF YouTube — Distraction-Free YouTube

A simple, lightweight multi-page website that lets you watch YouTube videos across curated categories — without the noise of ads, recommendations, or sidebar distractions.

---

## 🌐 Live Demo

> Open `index.html` in your browser to get started.

---

## 📁 Project Structure

```
Distraction_Free_YouTube_Using_Only_HTML/
├── index.html       # Home page with category selection
├── music.html       # Music videos page
├── gaming.html      # Gaming videos page
└── coding.html      # Coding/programming videos page
```

---

## ✨ Features

- 🎵 **Music** — Curated music and NCS copyright-free mixes
- 🎮 **Gaming** — Fun gaming videos and console content
- 💻 **Coding** — Programming tutorials (Python, C++, and more)
- 🔙 **Easy Navigation** — One-click back button on every category page
- 📱 **Responsive** — Works on desktop and mobile browsers

## 🚀 Getting Started

### Option 1: Open Locally

1. Clone or download this repository:
   ```bash
   git clone https://github.com/rr-saty/Distraction_Free_YouTube_Using_Only_HTML.git
   ```
2. Open `index.html` in any modern web browser.

> No build tools, no dependencies, no setup required.

### Option 2: Deploy on GitHub Pages

1. Push the project to a GitHub repository.
2. Go to **Settings → Pages**.
3. Set the source to the `main` branch and `/ (root)`.
4. Your site will be live at `https://rr-saty.github.io/Distraction_Free_YouTube_Using_Only_HTML/`.

---

## 🛠️ Built With

- **HTML5** — Structure and embedded YouTube iframes
- **No CSS framework** — Pure browser defaults (minimal and fast)
- **No JavaScript** — Zero dependencies

---

## 📌 How to Add More Videos

1. Open the relevant category file (e.g., `coding.html`).
2. Copy an existing `<iframe>` block.
3. Replace the `src` URL with your new YouTube embed link.
   - YouTube embed format: `https://www.youtube.com/embed/VIDEO_ID`
